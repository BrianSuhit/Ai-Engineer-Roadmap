# AI Engineer Roadmap (Interactive Visualization)

## 🎯 ¿Por qué este proyecto?
Este proyecto tiene como fin visualizar mi camino de aprendizaje hacia la Ingeniería en IA.

Basándome en el roadmap oficial de **roadmap.sh/ai-engineer**, he decidido crear un repositorio personal para replicarlo e ir completando cada nodo con mi propia investigación y proyectos personales que demuestren el conocimiento adquirido.

## 🚀 Objetivos de aprendizaje
*   **Dominio Técnico:** Adquirir el conocimiento completo de IA, las herramientas y metodologías que imperan en la industria actual.
*   **Registro Visual:** Llevar un registro práctico y visual de todo el conocimiento adquirido en mi camino como AI Engineer.
*   **Proyectos Reales:** Crear proyectos desde lo más básico hasta lo más avanzado, para entender qué tareas realizaré en un futuro entorno laboral.

## 🗺️ Estructura del Mapa
El roadmap se divide en un sistema jerárquico manual:
1.  **Nodos Principales:** El área general de un tema (Ej: Introducción).
2.  **Sub-nodos:** Puntos de conocimiento específico dentro de un nodo mayor.
3.  **Carpetas de Sub-nodos:** Agrupaciones de conceptos sueltos (Ej: "Common Terminology" dentro de Introducción).
4.  **Nodo Secundario (Proyecto):** Los nodos prácticos incluyen un nodo especial de "Proyecto" donde vuelco el código y la experiencia aprendida.
